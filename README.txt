- Please run app on port 6789

- When you are not logged in, you can only 'view' catalogs and items. As well as, access to JSON APIs.

- To modify items and catalogs, you can use you google account to register/login.

- You can only modify your own items and catalogs. You cannot create an item under someone else's catalog.

- You can use following links to get JSON outputs:

-- /catalog/JSON : All Catalogs
-- /catalog/catalog_id/item/JSON : All items under catalog with id = catalog_id
-- /catalog/item/JSON : All items
-- /catalog/catalog_id/item/item_id/JSON : Information about a specific item.
